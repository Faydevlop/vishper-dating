Voice-first social calling app UI prototype built with Expo Router and mock data.

## Commands

- `npm run start`
- `npm run android`
- `npm run androind` (typo-safe alias)
- `npm run android:device` (USB-connected device)
- `npm run androind:device` (typo-safe USB alias)
- `npm run android:usb` (alias for USB-connected device)
- `npm run android:apk:local` (local release APK build)
- `npx run android --device` (local wrapper that forwards to `expo run:android --device`)
- `npm run ios`
- `npm run web`
- `npm run typecheck`
- `npm run assets:generate`

## Windows Build Note

Android scripts and Gradle wrapper now force `GRADLE_USER_HOME` to `mobile/.g` to prevent Windows `Filename longer than 260 characters` errors during native builds.

## Assets

App icons and splash assets are generated by `scripts/generate-assets.ps1` into `assets/`.
